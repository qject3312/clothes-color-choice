# FitPick 핏픽 사용자 가이드

FitPick은 사용자가 보유한 옷을 등록하고, 등록된 옷 정보를 바탕으로 코디를 추천해주는 Python 기반 데스크톱 앱입니다.

사용자는 옷을 사진으로 등록하거나 직접 입력할 수 있으며, 앱은 옷의 종류, 색상, 특징, 온도, 사용자 선호 스타일 등을 고려하여 코디를 추천합니다.

1. 설치 및 실행 방법

6. 맞춤 코디 추천
7. 코디해보기
8. 온도별 추천

## 1. 설치 및 실행 방법

프로그램을 실행하려면 Python 실행 환경이 필요합니다.

프로젝트 폴더로 이동합니다.

```bash
cd clothes-color-choice
```

앱 폴더로 이동합니다.

```bash
cd clothing_app
```

프로그램을 실행합니다.

```bash
python main.py
```

정상적으로 실행되면 로그인 또는 회원가입 화면이 나타납니다.

---

## 2. 회원가입 및 로그인

### 2.1 회원가입

처음 사용하는 사용자는 회원가입을 진행합니다.

1. 로그인 화면에서 회원가입 버튼을 누릅니다.
2. 아이디와 비밀번호를 입력합니다.
3. 이름, 체형, 피부톤, 퍼스널 컬러, 선호 스타일 등의 정보를 입력하거나 선택합니다.
4. 회원가입을 완료하면 로그인 화면으로 이동합니다.

입력한 사용자 정보는 이후 맞춤 코디 추천에 활용됩니다.

### 2.2 로그인

1. 아이디와 비밀번호를 입력합니다.
2. 로그인 버튼을 누릅니다.
3. 로그인에 성공하면 홈 화면으로 이동합니다.

### 2.3 게스트 시작

계정을 만들지 않고 앱을 체험하고 싶은 경우 게스트 모드로 시작할 수 있습니다. 단, 사용자 맞춤 추천이나 저장 기능 일부는 제한될 수 있습니다.

---

## 3. 홈 화면 안내

홈 화면에서는 앱의 주요 기능으로 이동할 수 있습니다.

| 메뉴 | 설명 |
|---|---|
| 옷 등록 | 사진 또는 직접 입력으로 옷을 등록합니다. |
| 내 옷장 | 등록한 옷을 확인하고 수정 또는 삭제합니다. |
| 맞춤 추천 | 등록된 옷과 사용자 정보를 바탕으로 코디를 추천합니다. |
| 코디해보기 | 원하는 옷을 직접 선택하여 코디를 구성합니다. |
| 온도별 추천 | 입력한 기온에 맞는 옷 조합을 추천합니다. |
| 오늘의 추천 | 사용자 정보와 스타일을 바탕으로 오늘의 코디를 추천합니다. |
| 저장 코디 | 저장한 코디를 확인하거나 삭제합니다. |
| 프로필 | 사용자 정보를 확인하고 수정합니다. |

---

## 4. 옷 등록 방법

옷은 사진 등록 또는 직접 입력 방식으로 등록할 수 있습니다.

### 4.1 사진으로 등록

1. 홈 화면에서 옷 등록 메뉴를 선택합니다.
2. 사진 등록 버튼을 누릅니다.
3. 등록할 옷 사진을 선택합니다.
4. 앱이 사진에서 대표 색상을 자동으로 추출합니다.
5. 옷의 종류, 세부 종류, 특징을 선택합니다.
6. 저장 버튼을 누르면 옷장에 등록됩니다.

사진 등록 시 PNG, JPG 형식의 이미지를 사용하는 것이 좋습니다.

### 4.2 직접 입력으로 등록

1. 옷 등록 화면에서 직접 입력을 선택합니다.
2. 옷의 카테고리를 선택합니다.
   - 상의
   - 하의
   - 아우터
   - 신발
   - 악세서리
3. 세부 종류를 선택합니다.
4. 색상을 선택하거나 RGB 값을 조정합니다.
5. 핏, 두께, 무드, 계절감 등의 특징을 입력합니다.
6. 저장 버튼을 누르면 옷장에 등록됩니다.

---

## 5. 내 옷장 보기

내 옷장 화면에서는 사용자가 등록한 옷을 확인할 수 있습니다.

확인 가능한 정보는 다음과 같습니다.

- 옷 사진
- 카테고리
- 세부 종류
- 색상
- 특징

카테고리와 세부 종류 필터를 사용하여 원하는 옷만 볼 수 있습니다. 등록한 옷은 필요에 따라 수정하거나 삭제할 수 있습니다.

---

## 6. 맞춤 코디 추천

맞춤 추천 기능은 사용자가 등록한 옷과 사용자 정보를 바탕으로 코디를 추천합니다.

추천에는 다음 요소가 반영됩니다.

- 색상 조합
- 입력한 온도
- 체형 정보
- 퍼스널 컬러
- 선호 스타일
- 상황
- 코디 완성도

추천 결과는 점수와 함께 표시되며, 추천 근거를 통해 왜 해당 코디가 추천되었는지 확인할 수 있습니다.

---

## 7. 코디해보기

코디해보기 기능에서는 사용자가 직접 옷을 선택하여 코디를 구성할 수 있습니다.

1. 상의를 선택합니다.
2. 하의를 선택합니다.
3. 필요하면 아우터, 신발, 악세서리를 선택합니다.
4. 구성한 코디를 저장할 수 있습니다.

상의와 하의는 필수 항목이며, 나머지는 선택 항목입니다.

---

## 8. 온도별 추천

온도별 추천 기능은 입력한 기온에 어울리는 옷 종류와 코디를 추천합니다.

| 온도 | 추천 예시 |
|---|---|
| 더운 날 | 반팔, 얇은 하의 |
| 선선한 날 | 긴팔, 셔츠, 가벼운 아우터 |
| 추운 날 | 니트, 후드티, 코트, 패딩 |

등록된 옷이 부족하면 추천 결과가 제한될 수 있습니다.

---

## 9. 오늘의 추천 및 저장 코디

오늘의 추천 기능은 사용자의 퍼스널 컬러, 선호 스타일, 등록된 옷 정보를 바탕으로 자동 코디를 추천합니다.

마음에 드는 코디는 저장할 수 있으며, 저장 코디 화면에서 다시 확인하거나 삭제할 수 있습니다.

---

## 10. 사용자 정보 관리

프로필 화면에서는 사용자 정보를 확인하고 수정할 수 있습니다.

수정 가능한 정보 예시는 다음과 같습니다.

- 이름
- 체형
- 피부톤
- 퍼스널 컬러
- 선호 스타일

수정된 정보는 이후 맞춤 추천에 반영됩니다.

---

## 11. 주의사항

- 최소한 상의와 하의가 등록되어 있어야 코디 추천이 원활하게 작동합니다.
- 사진 등록 시 이미지 파일 경로가 변경되면 사진이 표시되지 않을 수 있습니다.
- 실행 환경에 따라 화면 크기나 버튼 모양이 다르게 보일 수 있습니다.
- 최종 실행 및 화면 확인은 팀에서 정한 실행 환경을 기준으로 진행하는 것이 좋습니다.
- 앱 실행 중 오류가 발생하면 프로그램을 종료한 뒤 다시 실행해보세요.
